- Async report generation queued, processing, failed, completed, and retry states.

### 12.3 UI Privacy Rules

- Public teaser screens must not show owner identity, full VIN, IC/passport, policy number, loan account, exact private document names, or raw record detail.
- Public report screens show immutable snapshot data only.
- Shared report expired/revoked states must be explicit and non-leaky.
- Dealer and insurer flows must clearly distinguish purchased report snapshots from raw owner data.
